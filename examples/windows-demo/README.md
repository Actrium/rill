# windows-demo

Windows example app demonstrating **rill sandbox** with `react-native-windows`.

## Configuration

| Item | Value |
|------|-------|
| Platform | Windows (`react-native-windows`) |
| Sandbox Engine | QuickJS |
| Host Runtime | Hermes |
| RN Version | 0.83+ |

## Quick Start

```powershell
cd examples/windows-demo

# 1. Run the bootstrap script (installs deps, generates RNW project, builds QuickJS)
powershell -ExecutionPolicy Bypass -File install.ps1

# 2. Start Metro bundler
npm start

# 3. Build & run (in another terminal)
npx react-native run-windows
```

## Manual Setup

```powershell
# 1. Install JS dependencies
npm install

# 2. Generate react-native-windows project (if windows/RillDemo.sln doesn't exist)
npx react-native-windows-init --overwrite --language cpp

# 3. Build QuickJS sandbox library
cmake -S windows/RillSandboxNative -B windows/RillSandboxNative/build -G "Visual Studio 17 2022" -A x64
cmake --build windows/RillSandboxNative/build --config Release

# 4. Open in Visual Studio and integrate native module
#    - Add RillSandboxNativeModule.cpp to the RillDemo project
#    - Link rill_quickjs_sandbox.lib
#    - Add include paths for QuickJS headers

# 5. Build & run
npx react-native run-windows
```

## Architecture

```
windows-demo/
├── package.json               # rill: "file:../../" (symlink to monorepo root)
├── metro.config.js            # Metro bundler config
├── babel.config.js            # Babel config
├── app.json                   # App registration
├── install.ps1                # Bootstrap script
├── windows/                   # Native Windows project
│   ├── RillDemo.sln           # Visual Studio solution (generated by RNW init)
│   ├── RillDemo/              # App project (generated)
│   └── RillSandboxNative/     # QuickJS sandbox native module
│       ├── RillSandboxNativeModule.h    # TurboModule header
│       ├── RillSandboxNativeModule.cpp  # QuickJS JSI binding installer
│       ├── CMakeLists.txt     # Build config for QuickJS static lib
│       └── pch.h              # Precompiled header
├── RCTNApps/                  # React Native JS source
│   ├── index.tsx              # App entry point (sandbox init)
│   ├── tsconfig.json
│   └── src/
│       ├── App.tsx            # Host app with Engine + Guest code
│       └── RillSandboxModule.ts  # TurboModule TS spec
└── TestCode/                  # JS performance benchmarks
    ├── fib.js
    ├── array.js
    ├── json.js
    └── string.js
```

## Dependencies

- Windows 10/11
- Visual Studio 2022 (Desktop C++ + UWP workloads)
- Windows 10/11 SDK
- Node.js >= 18
- CMake >= 3.16
- `rill` (resolved via `file:../../` symlink)
